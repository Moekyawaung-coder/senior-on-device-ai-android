# Senior On-Device AI Android Repository

Demonstrates how a Senior Android Engineer integrates on-device machine learning models into production apps while maintaining performance, privacy, and battery efficiency.

**Models Implemented:**
- Image classification with MobileNetV3
- Text summarization using Gemini Nano (on-device)
- Pose detection with MediaPipe
- Smart reply suggestions

**Tech Stack:** TensorFlow Lite, ML Kit, CameraX, Jetpack Compose, WorkManager for batch inference

**Linked to:** senior-kmp-full-template (for future KMP AI sharing) and senior-portfolio-v4.
